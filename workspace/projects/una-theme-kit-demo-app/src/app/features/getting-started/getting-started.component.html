Getting Started

A minimal CSS theme kit with variables, fluid typography, and zero dependencies.

1

Install

Add UNA Theme Kit to your project via npm:

npm install @heer.dev/una-theme-kit
2

Import Base CSS

Import the base styles in your main CSS file:

/* In your styles.css */
@import 'una-theme-kit/css/base.css';
3

Add Your Variables

Copy the CSS variables from starter.css into your stylesheet and customize them:

:root {{ '{' }}
  /* Layout */
  --max-width: 1280px;
  
  /* Typography */
  --family-1: system-ui, sans-serif;
  --font-size-0: clamp(0.75rem, 0.4vw + 0.65rem, 0.9rem);
  --font-size-1: clamp(0.875rem, 0.45vw + 0.75rem, 1rem);
  --font-size-2: clamp(1rem, 0.5vw + 0.85rem, 1.125rem);
  
  /* Spacing */
  --space-1: 0.5rem;
  --space-2: 1rem;
  --space-3: 2rem;
  
  /* ... and more */
{{ '}' }}

body {{ '{' }}
  /* Light theme colors */
  --color-1: hsl(210, 100%, 50%);
  --background-1: hsl(0, 0%, 96%);
  --background-2: hsl(0, 0%, 100%);
  --on-background: hsl(0, 0%, 15%);
  --border-1: hsl(0, 0%, 88%);
{{ '}' }}

Tip: See all available variables on the Variables page

4

Start Using

Use the CSS variables in your components:

.my-component {{ '{' }}
  background: var(--background-2);
  color: var(--on-background);
  padding: var(--space-2);
  border: 1px solid var(--border-1);
  border-radius: 8px;
{{ '}' }}
